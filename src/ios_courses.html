Course Structure

Swift basics and App Interfaces
Logic, Interaction
Working with Data
Submitting your app

You’ll learn

Swift basics and App Interfaces
(2 weeks)
The course includes:
  • The basics of Cocoa Framework
  • Swift code principles
  • Xcode introduction
  • IOS SDK overview
Logics, Interaction
(2 weeks)

We develop a back-end web application and APIs using a modern programming language and MVC web development framework.

The course includes:

Model-View-Controller Patterns, Navigation Controllers, UIKit frameworks, Interactions & Gestures, Delegation, Interaction Patterns, Core ios7 Libraries, Using most popular APIs.

Working with Data
(2 weeks)

We’ve just learned how to write an app with a simple user interface and basic behaviour. At this point, we are learning how to implement additional behavior that will turn a project into a full-featured app

The course includes:

Connection to the Internet. NSURLConnection. Core Data. Foundation Framework

Submitting your app
(2 weeks)

Working prototype is ready. Debugging and testing are completed. What’s next? We distribute apps through the stores, get using A/B testing processes and test different methods to attracting first users.

The course includes:

Unit and Integration testing. Defining Metadata. Submitting app. Release. A/B Testing. Attracting first traffic to application./p>
